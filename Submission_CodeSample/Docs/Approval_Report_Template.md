# Approval Report Template

작업 완료 후 사용할 보고 템플릿입니다.

```text
Approval Report

Objective:

Plan Executed:

Changed Files:

Implementation Summary:

Verification:

Manual Test Steps:

Risks / Notes:

Next Suggested Task:
```

## 작성 규칙

- 실제로 변경한 파일만 적습니다.
- 실제로 실행한 검증만 성공으로 보고합니다.
- 실행하지 못한 검증은 이유를 설명합니다.
- 남은 risk와 다음 작업을 구체적으로 적습니다.
